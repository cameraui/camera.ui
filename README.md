A self-hosted platform for your security cameras — live viewing, 24/7 recording\*, on-device AI detection, semantic search\*, smart-home integration, and push notifications\*, all built on an extensible plugin ecosystem for endless possibilities. Runs on hardware you own — no mandatory cloud, your footage stays with you.

## Getting Started

Please follow the documentation at [docs.cameraui.com](https://docs.cameraui.com) — it covers [installation](https://docs.cameraui.com/install/) (desktop app, Docker, Proxmox, bare-metal Linux, mobile apps), a guided [first run](https://docs.cameraui.com/intro/getting-started), and everything beyond. A [live demo](https://demo.cameraui.com) is in the works — coming soon.
